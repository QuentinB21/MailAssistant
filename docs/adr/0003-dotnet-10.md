# ADR 0003 — cibler .NET 10

- Statut : accepté
- Date : 2026-06-24

## Contexte

Le socle initial ciblait .NET 8. Le propriétaire du projet demande
explicitement l’utilisation de .NET 10, déjà installé dans l’environnement de
développement.

## Décision

Tous les projets backend ciblent `net10.0`. Le SDK est verrouillé sur
`10.0.100` avec mise à niveau automatique vers les correctifs compatibles.
Les packages `Microsoft.Extensions.*` sont alignés sur la génération 10.

## Conséquences

- Les développeurs et la CI doivent disposer du SDK .NET 10.
- Les fonctionnalités et performances de .NET 10 peuvent être utilisées.
- Le déploiement devra fournir un runtime .NET 10 compatible.
